# Blog Discoverability: RSS, Sitemap, SEO Metadata

**Date**: 2026-03-11
**Base URL**: `https://ogkranthi.github.io`
**Twitter**: `@ogkranthi`

## Goal

Add five discoverability features to the static blog in one batch: RSS feed, sitemap, robots.txt, Twitter card meta tags, JSON-LD structured data, and canonical URLs. RSS and sitemap are auto-generated via GitHub Action; the rest are one-time additions to existing files plus updates to copilot-instructions for future posts.

## Components

### 1. RSS Feed (`feed.xml`)

- RSS 2.0 XML generated by the update script
- Source: post cards parsed from `index.html` (title, excerpt, date, href)
- Only includes posts with real hrefs (skips `#` placeholders)
- Title + excerpt format, links back to full article
- Channel metadata: blog title, description, link, language

### 2. Sitemap (`sitemap.xml`)

- Auto-generated by the update script
- Lists `index.html` plus all post HTML files that exist on disk
- `<lastmod>` derived from file modification time
- Standard sitemap protocol format

### 3. `robots.txt`

- Static file, created once
- Allows all user-agents
- Points to `https://ogkranthi.github.io/sitemap.xml`

### 4. Twitter/X Card Meta Tags

Added to `<head>` of all HTML files (index + 5 posts):

```html
<meta name="twitter:card" content="summary">
<meta name="twitter:site" content="@ogkranthi">
<meta name="twitter:title" content="...">
<meta name="twitter:description" content="...">
```

Values sourced from existing `og:title` and `og:description` tags in each file.

### 5. JSON-LD Structured Data

**index.html** — `WebSite` schema:

```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Kranthi Manchikanti",
  "url": "https://ogkranthi.github.io",
  "description": "Technical deep-dives on AI Agents, AI Architectures, Reinforcement Learning, HITL frameworks, and AI safety."
}
```

**Post pages** — `Article` schema:

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "...",
  "author": { "@type": "Person", "name": "Kranthi Manchikanti" },
  "datePublished": "2026-...",
  "description": "..."
}
```

Values sourced from existing `<title>`, `<meta name="description">`, and article-meta sections.

### 6. Canonical URLs

Added to `<head>` of every page:

```html
<link rel="canonical" href="https://ogkranthi.github.io/page.html">
```

### 7. `index.html` Updates

- Add RSS autodiscovery link in `<head>`: `<link rel="alternate" type="application/rss+xml" ...>`
